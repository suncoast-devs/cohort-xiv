Git Cheet Sheet

here are some useful git commands for a new developer

  • > git add .

    Tells git that there are changes to track in the current directory. Also adds any new files to the current git repository.

  • > git commit

    this is a thing

    Creates a new save point for your code for you to go back. This requires `-m` flag to give the commit a meaningful message. This message shoudl be longer than a word but less than a novel.
